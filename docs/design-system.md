# VoteWave Design System

## Personality
Premium, fast, youthful, trustworthy, award-night energy.

## Colors
- Background: Deep black/navy `#070711`
- Card: Transparent glass white
- Primary accent: Gold `#f8c34a`
- Secondary accent: Royal purple `#6d28d9`
- Energy accent: Cyan `#22d3ee`

## Mobile Design Rules
- Bottom navigation on mobile.
- Large touch targets.
- Circular nominee images.
- Lazy-loaded images.
- Progress bars without public vote totals.
- No heavy UI library.

## Recommended Image Sizes
- Nominee images: square 500x500, compressed before upload.
- Event covers: 1200x700, compressed.
